# relatorios-fapesp-dados-lidar
Relatórios técnicos descritivos e produtos LiDAR (MDT, MDS, CHM e limites KML) das Unidades de Conservação do Estado de São Paulo cadastradas no sistema da Secretaria de Meio Ambiente, Infraestrutura e Logística. Projeto de Treinamento Técnico Nível 4 (TT-4), Processo FAPESP nº 24/15211-1.
